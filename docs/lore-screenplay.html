Acte I — Première connexion

Le joueur arrive sur le hub. ARCH établit le contrat : elle conseille, Node-0 exécute.

RELAY // ADVISORY CHANNEL

ARCH

ARCH

I'm ARCH — Triage & Recovery Advisory Core Entity. I kept your thread isolated. Listen — we don't have much time.

ARCH (suite)

Corrupted processes are flooding the system. The Breach is rising. I isolated Node-0 in a quarantined execution thread — the last bubble still stable inside a dying machine.

Pourquoi cette scène est là Poser ARCH en une phrase : nom, rôle, urgence. Annoncer la quarantaine sans cinématique. Le joueur comprend qu'il est dans le système, pas devant un menu abstrait. Le ton est professionnel, pas mascotte — on installe la relation advisory / exécution dès la première carte.

Quarantined Thread

ARCH

You are Node-0 — the last stable core, locked in quarantine. This thread is all that remains. Corrupted processes hammer the quarantine wall. Every impact and every second feeds the Breach closer to Meltdown.

Pourquoi cette scène est là Verrouiller le canon : le joueur est Node-0, pas un opérateur externe. Introduire Meltdown comme conséquence narrative de l'inaction et des impacts. Prépare la jauge Overload sans encore montrer le HUD combat.

Écran :
MENU
Affichage :
Spotlight sur bouton Start Run
Déclencheur :

RELAY // ADVISORY CHANNEL

Contain the Breach

ARCH

Stop corrupted processes before they break quarantine. Launch a Run — purge in the arena, salvage Shards from every kill. When the thread ends, spend them on the Skill Tree. I advise. You execute. Each upgrade buys us time.

Pourquoi cette scène est là Expliquer la boucle roguelite entière en un bloc : run → shards → skill tree. La phrase « I advise. You execute. » est le pilier de tout le dialogue ARCH — elle justifie pourquoi elle parle mais ne joue pas. Le spotlight sur Start Run convertit la lore en action immédiate.

ARCH

Purge Zone

ARCH

You're live. Your purge zone is the only weapon inside quarantine. Hold it over corrupted processes to destroy them. Every purge hit and every passing second feeds Overload.

Pourquoi cette scène est là Transition menu → action. « You're live » marque le basculement. Lie l'arme unique (purge) au coût (Overload) pour que le joueur comprenne le trade-off dès le premier contact — pas de tutoriel projectile classique.

Overload

ARCH

Overload is the live Breach meter on your thread. Keep it under control. At 100% Overload — Meltdown. The quarantine fails. Contain the threat before quarantine breaks.

Pourquoi cette scène est là Nommer la jauge HUD avec le lore Breach. Donner une condition de défaite claire (100 %). « Contain the threat » répète la fantasy victoire sans encore nommer le boss — le joueur sait pourquoi il doit agir vite (runs courtes, pression passive).

Run Shards

ARCH

Run Shards are stable data fragments salvaged from corrupted processes during a run. When a run ends, Shards transfer to your vault — currency for repairs between threads.

Pourquoi cette scène est là Connecter chaque kill à la progression permanente. Le mot « salvaged » renforce le fantasy terminal / récupération de données, pas or fantasy. Prépare le retour hub sans parler encore d'Anchor Fragments (réservés au boss).

ARCH

Overclock surges purge output — but Overload builds faster while active. Press Space when you can handle the pressure.

Annexe — Voix système (non ARCH)

Textes UI qui portent le lore sans être des répliques. Utiles pour cohérence VO / localisation future.

UI · MAIN_MENU

Menu titre

SYSTÈME

Zero Archive
The Zero Archive collapses under a wave of corrupted processes.
NODE-0 STATUS: UNSTABLE
You are Node-0 — a child process in quarantine.

Pourquoi c'est là Annonce le fantasy avant ARCH. « UNSTABLE » prépare le ton terminal. Pas de dialogue — pure UI diegetic.

UI · PAUSE

Pause combat

SYSTÈME

SYSTEM HALT — Node-0 execution suspended

Pourquoi c'est là Pause = arrêt d'exécution du thread, pas « menu pause » générique. Renforce l'identité Node-0.

UI · NODE0_BOOT · upgrade copy

Nœud racine — Node-0 Boot

SYSTÈME (tooltip)

Node-0 Boot — Initialize the quarantine thread — +1 purge hit damage

Pourquoi c'est là Seul module visible au départ (v4). Le texte fait double duty : lore (boot du thread) + feedback gameplay immédiat. Pas de voix ARCH — le joueur est l'acte d'initialisation.

UI · PLACEHOLDER

Modules réservés

SYSTÈME (tooltip)

Reserved module — Design slot — replace this node in a future update.
Module pending (placeholder_XX)

Pourquoi c'est là Honnête avec le joueur early access : slots visibles mais vides. Évite la frustration « bug » sur nœuds gris. ID visible pour communication PO / agent.
